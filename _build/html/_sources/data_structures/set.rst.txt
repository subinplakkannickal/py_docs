Set
===============

**Set Constructor:**

    * `a_set = {obj1, obj2, obj3, ...}`
    * `a_set = set(obj)`

* Unordered collection of unique data
* Mutable

**Other functionalities:**

    * `add(obj)`

        * Add an element to set
    
    * `union(set)`

        * `a_set.union(b_set)`, Return set of union of `a_set` and `b_set`
    
    * `intersection(set)`

        * `a_set.intersection(b_set)`, Return set of common elements of `a_set` and `b_set`
    
    * `difference(set)`

        * `a_set.difference(b_set)`, Return set of elements of `a_set` not in `b_set`
    
    * `symmetric_difference(set)`

        * `a_set.symmetric_difference(b_set)`, Return set of elements of `a_set` not in `b_set` and elements of `b_set` not in `a_set`.
